Common Questions

Does it work on Mac and PC?

Yes. We support macOS (M1/M2/M3 chips recommended) and Windows 10/11 with NVIDIA GPUs.

Does it use the internet?

The core processing happens locally on your machine. Internet is only used for updates or if you explicitly enable web search features.

Can I remove it later?

Absolutely. Nexus Personal is installed as a standard application container. Deleting it removes all system files instantly.

Is my data used for training?

No. Since the model runs locally, your data never leaves your device. We have no access to your files.
